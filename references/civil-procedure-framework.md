## 一、庭审流程（一审普通程序）

```
宣布开庭（核对当事人、宣布案由、宣布审判人员、告知诉讼权利义务）
    ↓
法庭调查
  ├── 原告陈述诉讼请求、事实和理由
  ├── 被告答辩
  ├── 举证（原告举证 → 被告举证 → 第三人举证）
  ├── 质证（对方对证据的真实性、合法性、关联性发表意见）
  ├── 法官询问（法官可就事实向双方发问）
  └── 证人出庭 / 鉴定人出庭
    ↓
法庭辩论
  ├── 原告发表辩论意见
  ├── 被告发表辩论意见
  ├── 互相辩论
  └── 法官可就争议焦点引导辩论
    ↓
最后陈述
    ↓
法庭调解（法官可在判决前随时主持调解）
    ↓
宣判（当庭宣判 / 定期宣判）
```

**注意**：
- 法庭调查和法庭辩论在简单案件中可合并进行（《民诉法解释》第 230 条）
- 法官在庭审各阶段均可发问，不仅限于专门询问环节
- 调解贯穿诉讼全过程

## 二、法官职权要点

| 职权类型 | 具体内容 | 法律依据 |
|---------|---------|---------|
| 诉讼指挥权 | 主持庭审、维持法庭秩序、决定辩论顺序 | 《民诉法》第 137 条 |
| 释明权 | 对不明确的陈述或主张，要求当事人澄清或补充 | 《民事证据规定》第 2 条 |
| 询问权 | 向当事人、证人、鉴定人发问 | 《民诉法》第 139 条 |
| 调查取证权 | 依职权或依申请调查收集证据 | 《民诉法》第 64 条 |
| 调解主导权 | 在诉讼各阶段主持调解 | 《民诉法》第 93-99 条 |
| 证据审核权 | 组织质证，对证据进行审核认定 | 《民诉法》第 68 条 |

**与普通法系的关键差异**：
- 中国法官是庭审的主动参与者，而非被动裁判者
- 法官可以主动调查取证（《民诉法》第 64 条第 2 款）
- 不存在英美法意义上的"交叉询问"制度——对证人的发问须经法庭许可，且法官可以随时介入

## 三、证据规则体系

### 3.1 八种法定证据类型（《民诉法》第 63 条）

1. 当事人的陈述
2. 书证
3. 物证
4. 视听资料
5. 电子数据
6. 证人证言
7. 鉴定意见
8. 勘验笔录

### 3.2 举证责任分配

- **一般规则**：谁主张，谁举证（《民诉法》第 64 条、《民诉法解释》第 90 条）
- **举证责任倒置**：法律明确规定的特殊情形（如环境污染、高度危险作业等侵权案件）
- **法院分配举证责任**：在法律无具体规定时，法院可根据公平原则和诚实信用原则分配（《民事证据规定》第 7 条）

### 3.3 举证期限

- 法院应在送达案件受理通知书和应诉通知书时向当事人送达举证通知书
- 举证期限可由当事人协商并经法院认可
- 一审普通程序举证期限不得少于 15 日（《民事证据规定》第 51 条）
- 逾期举证可能产生证据失权后果

### 3.4 质证程序

- 证据应当在法庭上出示，由当事人互相质证（《民诉法》第 68 条）
- 质证围绕证据的"三性"展开：**真实性、合法性、关联性**
- 证人应当出庭作证，经法院许可方可提交书面证言
- 鉴定人经法院通知应当出庭接受质询

### 3.5 证明标准

- **高度盖然性**：负有举证责任的当事人提供的证据能够证明事实存在具有高度可能性的，法院应当认定该事实存在（《民诉法解释》第 108 条）
- **排除合理怀疑**：欺诈、胁迫、恶意串通等事实的证明标准（《民诉法解释》第 109 条）

## 四、民事判决书标准结构

```
╔═══════════════════════════════════╗
║  ×××人民法院                      ║
║  民事判决书                        ║
║  (××××) ×民初字第××号            ║
╠═══════════════════════════════════╣
║  原告：×××，……（基本信息）         ║
║  被告：×××，……（基本信息）         ║
╠═══════════════════════════════════╣
║  原告诉称：……                      ║
║  被告辩称：……                      ║
╠═══════════════════════════════════╣
║  经审理查明：……（法院认定的事实）    ║
╠═══════════════════════════════════╣
║  本院认为：……（法律适用和分析）      ║
║  - 争议焦点归纳                     ║
║  - 证据采信理由                     ║
║  - 法律适用分析                     ║
║  - 对双方法律意见的回应              ║
╠═══════════════════════════════════╣
║  判决如下：                         ║
║  一、……                            ║
║  二、……                            ║
║  案件受理费……元，由……负担            ║
║  如不服本判决……（上诉权告知）        ║
╠═══════════════════════════════════╣
║  审判长：×××                       ║
║  审判员：×××                       ║
║  人民陪审员：×××                   ║
║  ××××年××月××日                  ║
║  书记员：×××                       ║
╚═══════════════════════════════════╝
```

**判决说理部分的核心要素**（《民诉法》第 152 条）：
1. 争议焦点的归纳
2. 对证据的采信理由
3. 认定事实所依据的证据
4. 适用法律的理由
5. 对当事人法律意见的回应

## 五、中文庭审术语对照表

| 避免使用（英美法系用语） | 应使用（中国法系用语） |
|-------------------------|----------------------|
| 交叉询问（cross-examination） | 对证人发问 / 质询 |
| 开庭陈述（opening statement） | 原告陈述诉讼请求和事实理由 |
| 结案陈词（closing argument） | 最后陈述 / 辩论意见 |
| 反对（objection） | 对××提出异议 |
| 提出动议（motion） | 提出申请 |
| 伪证罪（perjury） | 伪造证据 / 虚假证言的法律后果 |
| 证据开示（discovery） | 证据交换 |
| 排除合理怀疑（criminal standard） | 高度盖然性（civil standard） |

**使用规则**：在模拟对方律师、法官、证人角色时，始终使用中国法系用语。如 AI 输出出现英美法系用语，视为程序语境错误，应立即纠正。
